# Sistema de E-commerce

Projeto de e-commerce desenvolvido para gerenciamento de produtos, clientes e pedidos.
## Funcionalidades

* Cadastro de produtos
* Gerenciamento de clientes
* Carrinho de compras
* Controle de pedidos

## Instalação
1. Clone o repositório:
```bash
git clone <url-do-repositorio>
```

2. Instale as dependências:
```bash
npm install
```

3. Execute o projeto:
```bash
npm start
```

## Testes
Para executar os testes:
```bash
npm test
```
